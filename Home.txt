====== 学习规划 ======
Created Sunday 25 November 2018

===== 网络安全 =====
* 掌握重要工具
* 多动手复现漏洞
* 多做 CTF
* 看书看文章，做好笔记

===== 编程能力 =====
* 先学好 Python，多用 Python 写工具或者扩展脚本
* 学习 C 语言，了解操作系统和内存这些底层原理
* Java 有空也捡起来练练
* 要深入学习，不能停留于“使用”，代码要写整洁

===== Linux =====
* 掌握重要命令
* 记录好遇到的坑
* 深入了解系统工作原理

===== 课程 =====
* 找视频补一补概统
* 数据库要自己学
* 尽量应付一下作业
