====== 2to3 ======
Created Monday 26 November 2018

===== 主要变化 =====
* print 变成 print()
* 默认字符串为 Unicode 编码
* 只有新型类，没有经典类
* 更新异常语法，新增 as 关键字
* 更新整数：取消 long 型、除法保留小数、新的二进制和八进制字面量
* 提倡使用迭代器，节省内存资源

===== 转换工具 =====
* 警告：-3 开关
* 2to3 工具

===== 其他 =====
* Python3 中没有了 StringIO 和 cStringIO ，而使用 io.StringIO 和 io.BytesIO 分别处理文本和字节数据
* Python2 的 httplib 在 Python3 中是 http.client
* urlparse 在 Python3 中是 urllib.parse
* Python3 的 urlretrieve 方法在 urllib.request 模块中
* Mechanize 模块只支持 Python2.x，在 Python3 中可以用 [[https://mechanicalsoup.readthedocs.io/en/stable/introduction.html|MechanicalSoup]] 代替
