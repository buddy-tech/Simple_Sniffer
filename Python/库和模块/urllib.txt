====== urllib ======
Created Monday 26 November 2018

Python2 中的 urllib、urllib2 等模块在 Python3 都统一到 urllib 中。

* urllib.request：是一个用于获取URL 的模块，可以完成网页请求和下载等操作。
* urllib.error：包含了由 request 引起的异常
* urllib.parse：解析 URL
* urllib.robotparser：分析 robots.txt 文件

文档：https://yiyibooks.cn/xx/python_352/library/urllib.html
